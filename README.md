* **Teaching Web3 smart contract development** using **Plutus** and **Haskell**
* **Creating blockchain assets** (FTs & NFTs)
* **Understanding key Cardano technologies** like CIP-1694, Midnight, and Midgard
* **Building functional programming and problem-solving skills**
* **Contributing to open source** via GitHub (forks, PRs, docs)
* **Promoting Cardano community organizations** like Cardano Foundation, IOG, Intersect, Emurgo, and Gimbalabs
* **Getting live mentorship** from the Intersect MBO Developer Advocate Program
